README
- Each face recognition system has its own folder
- Under these systems, there is a file named system.py which is responsible for calculating the mated and non-mated scores and storing them in text files
- Under the DET folder, there is an ipynb file that shows the different DET plots for the different systems considered. 
- the papers that were referenced are included in the research papers folder. 
The mated and non_mated txt files of each system can be found under the DET folder.
